INTRO VIDEO (plays first, once, before the photo slideshow)
==============================================================

This folder holds the video that plays first at the very top of
the website, before the photo slideshow starts.

HOW TO CHANGE THE VIDEO:
- Upload your new video into this folder named exactly: intro.mp4
  (replace the existing file with the same name -- no code changes
  needed).

RULES / RECOMMENDATIONS:
- Format: .mp4 (H.264 video codec) works everywhere.
- The video plays automatically, muted, and only once -- then the
  photo slideshow takes over automatically. Whatever length your
  video is, that's how long it plays before switching to photos.
- Ratio: 16:9 (widescreen), same as the slideshow photos.
- Keep the file small for fast loading on mobile -- ideally under
  3-4MB. Use any online video compressor if your file is bigger
  (e.g. HandBrake, or search "compress mp4 online").
- The video has no sound in the browser (autoplay video must be
  muted by browser rules), so audio isn't necessary in the file.
- If this file is ever removed, the site automatically skips
  straight to the photo slideshow -- nothing breaks.
