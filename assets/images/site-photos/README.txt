OTHER SITE PHOTOS
===================

These are the individual photos used on the service cards. Each file
has a fixed job -- to change a photo, replace the file with the SAME
NAME (keep the exact filename so the website keeps finding it).

FILES & WHERE THEY SHOW UP:

  card-digital-marketing-16x10.png   -> "Digital Marketing" service card
  card-ai-16x10.png                  -> "Artificial Intelligence" service card
  card-programming-16x10.png         -> "Programming Language" service card
  card-graphic-design-16x10.png      -> "Graphic Design" service card
  card-tally-16x10.png               -> "Tally Prime (GST + ERP)" service card
  card-ms-office-16x10.png           -> "MS Office & Office Automation" service card
  card-autocad-16x10.png             -> "AutoCAD" service card
      Ratio: 16:10 (landscape). ~1200x750px recommended.

HOW TO CHANGE A PHOTO:
- Rename your new photo to match the exact filename above
  (e.g. card-ai-16x10.png), then upload it into this folder to
  overwrite the old one. That's it -- no code changes needed.
- If your new file is a .jpg instead of .png (or vice versa), you
  MUST also update the matching <img src="..."> line in index.html
  to point to the new extension, or the photo won't show up.

File size: keep each photo under ~300KB for fast loading.

(The "Why FutureX" section no longer uses photos -- it now uses
simple line icons instead, so there's nothing to manage there.)
